判断题 (5题，每题2分，共10分)

1 缺省形参必须放在非缺省形参右侧。
2 函数重载时，返回值类型不同可构成重载。
3 常对象只能调用常成员函数。
4 虚基类构造函数优先于所有非虚基类执行。
5 静态成员函数内可使用 this 指针。

微型编程题 (2题，每题10分，共20分)

1 操作符重载

实现 Point 类的 + 运算符（成员函数）和 << 输出（友元函数）：

class Point { int x, y; public: Point(int x, int y) : x(x), y(y) {} // 补全运算符重载 ________ operator+(const Point& rhs) { return ________; } friend ostream& ________(ostream& os, const Point& p); }; // 补全友元函数 ostream& operator<<(ostream& os, const Point& p) { os << "(" << p.x << "," << p.y << ")"; return os; }
2 虚函数与多态

补全派生类对纯虚函数的重写：

class Shape { public: virtual void draw() = 0; // 纯虚函数 }; class Circle : public Shape { // 重写 draw() 函数，输出 "Drawing Circle" ________ void draw() ________ { ________ } };
